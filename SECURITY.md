# Security policy

Crabcraft parses untrusted server traffic and handles Minecraft session tokens.
Please do not open a public issue for vulnerabilities that could expose accounts,
execute code, exhaust resources, or bypass protocol bounds.

Report vulnerabilities through GitHub's private vulnerability reporting for this
repository. Include the affected commit, reproduction steps, impact, and any
suggested mitigation. Do not include live access tokens, private keys, or another
person's server data.

Only the latest commit on `main` is supported during this pre-release phase. We
will acknowledge a complete report when maintainers are available, investigate,
and coordinate disclosure after a fix is ready.
